# 仕様アーカイブ（index）

`docs-ja/spec/archive/` は、`docs-ja/spec/` から退役した旧仕様を保管する場所です。  
`docs-ja/spec/` 直下は常に現行仕様のみを置きます。

## 運用ルール

- 旧仕様へ移すときは、ファイル名を `YYYYMMDD-<slug>.md` 形式にします。
- 先頭に次の3点を明記します。
  - 元ファイル
  - 退役理由
  - 現行の参照先
- アーカイブ追加時は、この `index.md` にリンクを追記します。
