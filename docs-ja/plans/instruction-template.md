# 指示テンプレート（優先度上書き用）

最終更新: 2026-02-22

`docs-ja/todo.md` を正本としたまま、今回の作業で優先度を上書きしたいときに使うテンプレートです。  
`todo2.md` は使いません。

## テンプレート

```md
今回の対象ID:
- P1-COMP-01
- P1-COMP-02

完了条件:
- unit test が通る
- docs-ja の関連記述を更新する

非対象:
- docs/ の翻訳更新
- リファクタ全般

優先度:
- 今回限り最優先
```

## 運用ルール

- 優先度上書きはチャット指示で行う（`todo2.md` は使わない）。
- 指示では最低限「対象ID」「完了条件」「非対象」を明記する。
- 指示がない場合は `docs-ja/todo.md` の優先順で進める。
