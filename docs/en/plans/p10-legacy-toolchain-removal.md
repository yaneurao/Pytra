# P10-LEGACY-TOOLCHAIN-REMOVAL: Remove old toolchain + pytra-cli.py

Last updated: 2026-04-03

## Background

Pytra's transpile pipeline is migrating from the old toolchain (`src/toolchain/`) to the new toolchain2 (`src/toolchain2/`). The CLI has already migrated from the old `src/pytra-cli.py` to `src/pytra-cli2.py`.

Once all 18 language emitters have been implemented in toolchain2 (P1-*-EMITTER-S1 Completed), there will be zero remaining dependencies on the old pipeline and it can be deleted.

## Start Condition

**Start on the user's signal.** Execute only when all contributors have stopped.

## Deletion Targets

### `src/toolchain/` (old emitter + compile + frontends)

| Directory | Contents |
|---|---|
| `src/toolchain/emit/<lang>/` | Old emitter for each language (all 18 languages) |
| `src/toolchain/compile/` | Old compile path |
| `src/toolchain/frontends/` | Old CLI frontend (`transpile_cli.py`, etc.) |
| `src/toolchain/misc/` | Old EAST generation (`east.py`, etc.) |
| `src/toolchain/link/` | Old linker (if present) |

### `src/pytra-cli.py` (old CLI)

After deletion, rename `src/pytra-cli2.py` to `src/pytra-cli.py`.

### Reference updates

- `docs/ja/spec/` — Emitter guide already updated (old toolchain references removed 2026-04-03)
- `docs/ja/tutorial/` — References to `pytra-cli.py` → `pytra-cli2.py` (revert to `pytra-cli.py` after rename)
- `tools/run/run_local_ci.py` — Remove old pipeline references if any
- `tools/README.md` — Remove old pipeline references from the tool registry
- `test/` — Tests using the old pipeline (migrate or delete)
- `README.md` / `docs/ja/README.md` — Update CLI name

## Notes

- **Delete in one batch.** Do not remove individual old emitters incrementally — the risk of leaving dangling references is high.
- **Tag `git tag v0.x-pre-toolchain-removal` before deleting.** In case a rollback is ever needed.
- **Confirm no tests depend on the old toolchain before deleting.** Run `grep -r "from toolchain\." tools/ test/` to identify dependencies.
- **Regenerate selfhost golden files with toolchain2.** Confirm that no golden files generated by the old toolchain remain.
- **After renaming `pytra-cli2.py` → `pytra-cli.py`, update all document CLI names.** Replace every reference to `pytra-cli2` with `pytra-cli`.
